# Project 001: P0 핵심 이벤트 루프 구축

## Goal
2주 내에 KR/US 관심 종목 등록부터 급등락 감지, 원인 카드 생성, 알림 전달, 이벤트 히스토리 조회까지 P0 전체 흐름을 동작 가능한 MVP로 완성한다.

## Context
- 현재 저장소는 문서/하네스 중심의 초기 상태이며 구현 코드는 아직 없다.
- 백로그 항목인 `브로커 연동 기반 포트폴리오 자동 동기화`, `장전/장후 전용 알림 규칙 세분화`는 이번 프로젝트에서 미룬다.
- 이번 페이즈는 PRD의 P0 다섯 기능을 우선 구현하고, P1/P2 확장 기능은 제외한다.

## Scope
- 관심 종목 등록/삭제 API 구현 (KR/US, 중복 방지)
- 급등락 감지 엔진 구현 (임계값, 디바운스, UTC 저장)
- 원인 카드 후보 1~3개 생성 및 근거 URL 필수 노출 규칙 적용
- 인앱/이메일 알림 발송 파이프라인 및 30분 쿨다운 적용
- 최근 30일 이벤트 히스토리 목록/상세 조회 API 구현 (종목/날짜 필터, 오류 응답 포함)

## Out of Scope
- P1 전체: 사용자 임계값 커스터마이즈, 알림 품질 피드백, 장전/장후 라벨링 고도화
- P2 전체: 포트폴리오 연동, 모바일 푸시 알림, B2B 이벤트 API/위젯
- 자동매매/투자 추천 등 자문성 기능

## Definition of Done
- All features in features.json are "done"
- All tests pass (including all previous tests)
- No regressions
