# Project 003: 12-Week Intensive Product Hardening

## Goal
12주 동안 사용자 관점에서 체감되는 핵심 경험(입력 신뢰, 원인 신뢰, 요약 경험, 비교 해석)을 세밀하게 고도화해 베타를 실사용 단계로 끌어올린다.

## Context
- `project-002-production-beta`에서 핵심 플로우는 구현됐지만, 회고에서 UI 완성도·입력 검증·신뢰도 투명성이 부족하다는 피드백이 있었다.
- `2026-02-17` 조사 결과에서도 차별화 포인트는 "속도" 자체가 아니라 "신뢰 가능한 원인 설명"과 "행동 가능한 요약 경험"으로 확인됐다.
- 이번 프로젝트는 신규 시장 확장보다 사용자 신뢰와 사용성 완성도를 우선하며, 3개월 내 파일럿 품질 지표를 검증 가능한 수준으로 만든다.

## Scope
- 시드 카탈로그 동기화/검증 기반 관심종목 입력 신뢰 체계 확립
- 검색/선택 기반 관심종목 UX와 상태 화면(로딩/빈/오류/성공) 고도화
- 근거 URL 품질 게이트, confidence 분해 설명, 정정 이력까지 포함한 원인 카드 신뢰도 강화
- What changed 델타 재알림으로 알림 피로를 낮추고 변경점 전달력 강화
- 개장 전/장마감 브리프 생성·전달·UI 플로우 구현
- 상충 근거 비교 카드로 해석 편향을 줄이는 사용자 경험 추가
- E2E/시각 회귀/계약 테스트와 릴리즈 게이트를 통해 12주 계획을 안정적으로 운영

## Out of Scope
- 자동매매/주문 실행 및 투자자문 기능
- 신규 자산군(코인/원자재/채권) 확장
- 국가 추가(비 KR/US) 및 현지 규제 확장 대응
