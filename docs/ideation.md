# Ideation: KR/US Stock Event Monitor

## Core Idea
- 한국/미국 주식을 보유하거나 관심종목으로 등록한 사용자를 대상으로, 급등락 시점에 "왜 움직였는지"를 빠르게 설명해주는 서비스.
- 핵심 가치: 가격 알림이 아니라 **원인 중심 이벤트 요약**(실적/가이던스/공시/뉴스/섹터/거시 변수) 제공.
- 왜 지금: 한국 리테일 투자자 비중이 높고, 미국 주식 투자도 확대되면서 정보 탐색 피로가 커짐.

## Technical Feasibility
- **핵심 데이터 소스는 이미 성숙**:
  - 미국: SEC EDGAR API(인증키 없이 JSON 제공, 실시간 갱신).
  - 한국: OPEN DART(공시 API, 인증키 발급 기반).
  - 시세/체결/호가: 상용 API(예: Polygon REST/WebSocket).
- **구현 난이도**
  - MVP(알림 + 이벤트 수집 + 원인 후보 1~3개 제시): 8~12주
  - 신뢰도 높은 원인 추론(정확도/커버리지/지연 최적화): 추가 1~2개 분기
- **핵심 난제**
  - 인과 추정: 같은 시간대 복수 이벤트(실적+거시+수급) 분리
  - 타임라인 정합성: 거래시간/장전·장후/국가별 시차
  - 법적/컴플라이언스: 투자자문 오인 방지 문구와 근거 링크 설계

- Technical feasibility: 4

- Timing: 4
- Effort vs. impact: 3
- **Total: 18/25 → Conditional Go**

## Competitive Landscape
- 직접 경쟁군은 이미 강함(실시간 뉴스/알림/차트/리서치).
- 다만 "KR+US 동시", "한국어 원인 설명", "개인 포트폴리오 맥락"을 결합한 제품은 상대적으로 공백.

## Market Opportunity
- 한국: 리테일 투자자 기반이 매우 큼(약 1,400만명대).
- 미국: 성인 주식 보유 비중 62%(2025), 대형 브로커/플랫폼에서 계좌·거래 활동 증가.
- 현실적 초기 기회는 "모든 투자자"가 아니라 **이벤트 민감도가 높은 능동 투자층**.

## Risks (top 3 with mitigations)
1. **원인 오판으로 신뢰 하락**
- Mitigation: 단정형 문구 금지, "가능성 점수" + 근거 링크 + 대체 시나리오 2~3개 제공.

2. **데이터 비용/라이선스 부담**
- Mitigation: 초기엔 공시+무료/저가 데이터 조합으로 MVP 구축, 유료 실시간 피드는 활성 사용자 비율 기반으로 단계적 확장.

3. **경쟁 과밀로 유료 전환 저조**
- Mitigation: KR+US 동시 커버, 한국어 요약, 개인 포트폴리오 맥락(보유 비중/평단 영향)으로 명확한 포지셔닝.

## Recommendation (Go / Conditional Go / No-Go)
**Conditional Go**

실행 조건:
- 1단계는 "한국 투자자의 한국/미국 대형주"로 범위 제한(예: 상위 500~1,000 종목).
- "속도"보다 "설명 신뢰도" KPI 우선(정확도/근거 클릭률/알림 유지율).
- 4~6주 파일럿(50~150명)에서 유료 의향/재방문/알림 유지율 검증 후 본개발 확대.
